# GX教育项目交付管理系统

## 项目简介
教育信息化项目全生命周期管理平台，涵盖设备管理、任务计划、交付跟踪、培训管理、风险管理、报告生成等核心功能。

## 技术栈
- **前端**：React 18 + Ant Design + TypeScript
- **后端**：Python FastAPI + SQLAlchemy ORM
- **数据库**：MySQL 8.0
- **部署**：Docker + Ubuntu 24.04
