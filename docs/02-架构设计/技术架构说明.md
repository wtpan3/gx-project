# GX教育项目交付管理系统 - 技术架构说明

## 技术栈
| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | React 18 + TypeScript + Ant Design 5.x | UI框架 |
| 后端 | Python 3.11+ + FastAPI + SQLAlchemy | API框架 |
| 数据库 | MySQL 8.0（Docker容器） | 数据存储 |
| 部署 | Ubuntu 24.04 + Screen + Nginx | 运行环境 |

## 系统架构
- 前后端分离架构
- 后端提供 RESTful API，JWT 认证
- 前端通过 Axios 调用后端 API
- 数据库通过 SQLAlchemy ORM 访问

- RESTful风格
- JWT认证
- 统一响应格式
