## 核心表结构

### users - 用户表
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| username | VARCHAR(50) | 登录账号，唯一 |
| password_hash | VARCHAR(255) | 密码哈希（bcrypt） |
| real_name | VARCHAR(50) | 真实姓名 |
| role | VARCHAR(20) | admin/project_manager/campus_manager |
| phone | VARCHAR(20) | 手机号，唯一 |
| email | VARCHAR(100) | 邮箱 |
| status | VARCHAR(10) | 启用/停用 |
