### 1.3 连接池配置

后端 `database.py` 中配置连接池：

```python
from sqlalchemy import create_engine

engine = create_engine(
    DATABASE_URL,
    pool_size=10,          # 连接池大小
    max_overflow=20,       # 最大溢出连接数
    pool_timeout=30,       # 获取连接超时（秒）
    pool_recycle=3600,     # 连接回收时间（秒）
    pool_pre_ping=True     # 连接前检查有效性
)
```


## 二、后端优化

### 2.1 接口响应缓存

适用于不常变化的数据（如字典数据）：

```python
from functools import lru_cache

@lru_cache(maxsize=128)
def get_dict_items(category: str):
    return db.query(DictItem).filter(DictItem.category == category).all()
```

### 2.2 分页查询

所有列表接口必须支持分页：

```python
def get_task_list(page: int = 1, page_size: int = 20):
    offset = (page - 1) * page_size
    return db.query(Task).offset(offset).limit(page_size).all()
```

### 2.3 N+1 查询优化

使用 SQLAlchemy 的 `joinedload()` 预加载关联数据：

```python
from sqlalchemy.orm import joinedload

# 避免 N+1
tasks = db.query(Task).options(joinedload(Task.assignee)).all()
```

## 四、Nginx 优化

```nginx
# Gzip压缩
gzip on;
gzip_types text/plain text/css application/json application/javascript text/xml application/xml;

# 静态资源缓存
location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg)$ {
    expires 1y;
    add_header Cache-Control "public, immutable";
}

# 限流（防止恶意请求）
limit_req_zone $binary_remote_addr zone=api:10m rate=10r/s;
location /api/ {
    limit_req zone=api burst=20 nodelay;
    proxy_pass http://127.0.0.1:8000;
}
```
