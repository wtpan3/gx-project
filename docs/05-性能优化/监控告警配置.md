## 二、服务器监控（基础）

### 2.1 资源使用监控脚本

创建 `~/monitor/health_check.sh`：

```bash
#!/bin/bash
# 服务器基础监控脚本

THRESHOLD_CPU=80
THRESHOLD_MEM=80
THRESHOLD_DISK=85

# CPU 使用率
CPU_USAGE=$(top -bn1 | grep "Cpu(s)" | awk '{print $2}' | cut -d. -f1)
if [ $CPU_USAGE -gt $THRESHOLD_CPU ]; then
    echo "⚠️ CPU 使用率过高: ${CPU_USAGE}%"
fi

# 内存使用率
MEM_USAGE=$(free | grep Mem | awk '{printf "%.0f", $3/$2 * 100}')
if [ $MEM_USAGE -gt $THRESHOLD_MEM ]; then
    echo "⚠️ 内存使用率过高: ${MEM_USAGE}%"
fi

# 磁盘使用率
DISK_USAGE=$(df -h / | awk 'NR==2 {print $5}' | sed 's/%//')
if [ $DISK_USAGE -gt $THRESHOLD_DISK ]; then
    echo "⚠️ 磁盘使用率过高: ${DISK_USAGE}%"
fi
```

### 2.2 定时检查（crontab）


## 四、日志监控

### 4.1 常用日志命令

```bash

sudo journalctl -u gx-backend -f

# 错误日志过滤
sudo journalctl -u gx-backend -p err
```

### 4.2 日志轮转配置

```bash
# 配置 journald 日志上限
sudo tee /etc/systemd/journald.conf << 'EOF'
SystemMaxUse=500M
SystemMaxFileSize=100M
MaxRetentionSec=7day
EOF

sudo systemctl restart systemd-journald
```


## 五、性能监控

### 5.1 接口响应时间监控

在后端添加中间件记录响应时间：

```python
# app/middleware.py
import time
from fastapi import Request
import logging

logger = logging.getLogger(__name__)

@app.middleware("http")
async def log_requests(request: Request, call_next):
    start_time = time.time()
    response = await call_next(request)
    process_time = time.time() - start_time
    
    if process_time > 1.0:  # 超过1秒记录告警
        logger.warning(f"慢请求: {request.url.path} 耗时 {process_time:.2f}s")
    
    response.headers["X-Process-Time"] = str(process_time)
    return response
```

## 七、健康检查脚本（一键运行）

```bash
cat > ~/gx-project/scripts/health-check.sh << 'EOF'
#!/bin/bash
echo "========================================="
echo "        GX项目 系统健康检查"
echo "        $(date)"
echo "========================================="

# 1. 后端检查
echo -n "后端服务 .......... "
curl -s http://127.0.0.1:8000/health > /dev/null
if [ $? -eq 0 ]; then
    echo "✅ 正常"
else
    echo "❌ 异常"
fi

# 2. 前端检查
echo -n "前端服务 .......... "
curl -s http://127.0.0.1/ | head -1 | grep -q "<!DOCTYPE"
if [ $? -eq 0 ]; then
    echo "✅ 正常"
else
    echo "❌ 异常"
fi

# 3. MySQL检查
echo -n "MySQL容器 ........ "
docker ps | grep gx_mysql > /dev/null
if [ $? -eq 0 ]; then
    echo "✅ 正常"
else
    echo "❌ 异常"
fi

# 4. 系统资源
echo "-----------------------------------------"
echo "CPU 使用率: $(top -bn1 | grep 'Cpu(s)' | awk '{print $2}' | cut -d. -f1)%"
echo "内存使用率: $(free | grep Mem | awk '{printf "%.0f", $3/$2 * 100}')%"
echo "磁盘使用率: $(df -h / | awk 'NR==2 {print $5}')"
echo "-----------------------------------------"
echo "系统负载: $(uptime | awk -F'load average:' '{print $2}')"
echo "========================================="
EOF

chmod +x ~/gx-project/scripts/health-check.sh
```
