# 首页原型变更记录

## 版本历史

### V2.1 - 2026-07-20 第八次修订

**修改内容**：
1. **顶部卡片区横向滚动** - 7 个卡片外层加 `.stat-scroll` 容器(overflow-x:auto),内层 min-width 980px、单卡 min-width 120px;宽度不足时整体横向滚动,不再挤压卡片(解决"查看重点校 →"折行问题)
2. **交付进度区横向滚动** - 三栏(学校/硬件/软件)外层加 `.progress-scroll` 容器,内层 min-width 880px、单栏 min-width 280px;解决硬件环形图 legend 数字溢出、软件"已上线"标签折行问题
3. **软件进度标签防折行** - soft-item 顶部行加 `white-space:nowrap`,模块名超长省略号截断,状态标签 `flex-shrink:0` 不被压缩

**同步文档更新**：
- **代码规范 `3.6 UI 布局与表格规范（强制）`** - 新增章节,将"超宽横向滚动、不压缩不换行"规则固化为开发规范:容器溢出规则、表格规范(table-layout:fixed + 固定列宽 + 省略号 + Tooltip + scroll.x)、卡片/多栏布局规则,并给出 CSS 与 Ant Design Table 示例代码,后续所有页面开发均须遵循

### V2.1 - 2026-07-20 第六次修订

**修改内容**：
1. **里程碑表格列宽固定化**
   - 阶段列: 150px (至少5个中文字符)
   - 任务列: 200px (至少9个中文字符)
   - 甘特图列: 占剩余宽度,最小 400px
   - 开始/结束日期列: 各 140px (完整显示 yyyy-mm-dd)
   - 状态列: 120px (完整显示"已完成/进行中/未开始/已延期")
2. **表格强制固定布局** - 采用 `table-layout: fixed`,不允许内容自动撑开列宽
3. **表格横向滚动** - 表格最小宽度 1150px,外层加 `.gantt-container` 容器设置 `overflow-x: auto`,允许横向滚动查看完整内容
4. **文本溢出处理** - 所有文本列保持 `text-overflow: ellipsis` + title 属性,超长内容截断,悬停显示全文

**同步文档更新**：
- 需求文档 `6.1.5` 里程碑区:补充完整的列宽规范(固定px值)、文本溢出处理规则、表格滚动规则、table-layout:fixed 强制固定布局说明
