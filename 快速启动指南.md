## 📊 API测试（使用curl或Postman）

### 1. 获取任务列表
```bash
curl "http://127.0.0.1:8000/api/v1/wbs-tasks?page=1&page_size=10"
```

### 2. 按状态筛选
```bash
curl "http://127.0.0.1:8000/api/v1/wbs-tasks?status=进行中"
```

### 3. 关键词搜索
```bash
curl "http://127.0.0.1:8000/api/v1/wbs-tasks?keyword=验收"
```

### 4. 获取详情
```bash
curl "http://127.0.0.1:8000/api/v1/wbs-tasks/34"
```

### 5. 创建任务
```bash
curl -X POST "http://127.0.0.1:8000/api/v1/wbs-tasks" \
  -H "Content-Type: application/json; charset=utf-8" \
  -d '{
    "construction_year": "2026",
    "project_phase_l1": "测试阶段",
    "sub_phase_l2": "功能测试",
    "task_package_l3": "API测试",
    "work_content_l4": "测试创建任务",
    "task_code": "TEST-999",
    "priority": "高",
    "status": "待开始",
    "plan_start_date": "2026-07-21",
    "plan_end_date": "2026-07-25",
    "responsible_person_id": 1,
    "school_id": 1
  }'
```

### 6. 更新任务
```bash
curl -X PUT "http://127.0.0.1:8000/api/v1/wbs-tasks/34" \
  -H "Content-Type: application/json; charset=utf-8" \
  -d '{
    "status": "进行中",
    "progress_note": "测试更新功能"
  }'
```

### 7. 删除任务
```bash
curl -X DELETE "http://127.0.0.1:8000/api/v1/wbs-tasks/36"
```

## 🔧 常见问题排查

### 问题1：后端启动失败

**症状：** `ImportError` 或 `ModuleNotFoundError`

**解决：**
```powershell
cd "f:\claude code\Projectmanage\gx-project\backend"
pip install -r requirements.txt
```

### 问题2：数据库连接失败

**症状：** `Can't connect to MySQL server`

**检查：**
1. 数据库地址：124.222.151.69:3306
2. 用户名/密码：root / GX2026!root
3. 数据库名：gx_project_dev
4. 网络连通性：`ping 124.222.151.69`

### 问题3：前端编译错误

**症状：** `Module not found` 或 TypeScript错误

**解决：**
```powershell
cd "f:\claude code\Projectmanage\gx-project\frontend"
npm install
```

### 问题4：中文乱码

**症状：** API返回的中文显示为乱码或问号

**检查：**
- 数据库连接字符集：`charset=utf8mb4`
- HTTP请求头：`Content-Type: application/json; charset=utf-8`
- 已修复：`backend/app/database.py` 已添加字符集配置

### 问题5：CORS跨域错误

**症状：** 浏览器控制台报 `CORS policy` 错误

**检查：** `backend/app/main.py` 中的CORS配置是否包含前端地址
